# Sistem Nilai Mahasiswa

## Deskripsi
Sistem Nilai Mahasiswa adalah aplikasi yang digunakan untuk mengelola data nilai mahasiswa. Aplikasi ini memiliki fitur login sederhana, CRUD data, dan pencarian data.

## Cara Instalasi
1. Download atau clone repository.
2. Buka project menggunakan Visual Studio 2019.
3. Pastikan database Microsoft Access tersedia pada folder project.
4. Jalankan project.

## Cara Menjalankan
1. Buka project di Visual Studio 2019.
2. Tekan tombol Start atau F5.
3. Login menggunakan akun yang tersedia.
4. Gunakan menu untuk mengelola data mahasiswa.

## Struktur Folder
- Form Login
- Form Menu
- Form Mahasiswa
- Form Nilai
- Database
- Laporan (jika ada)
